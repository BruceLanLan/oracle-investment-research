# 投资神谕研报系统（Oracle Investment Research）

### 多风格 · 全赛道 · 全阶段 · AI驱动的投资研究框架

---

## 🎯 核心定位

这是一个**投资方法论路由器**——根据你的需求，自动匹配最合适的投资分析框架。

```
你输入：分析苹果公司
      ↓
系统识别：成熟企业 + 二级市场 + 价值投资偏好
      ↓
路由至：巴菲特风格（styles/buffett/）
      ↓
输出：完整12模块深度报告
```

---

## 📊 支持的风格（投资人方法论）

| 风格 | 适用场景 | 核心框架 |
|------|---------|---------|
| **巴菲特** ⭐ | 成熟企业/二级市场/长期价值 | 护城河 + 所有者盈余 + 安全边际 |
| **彼得·蒂尔** ⭐ | 早期VC/创业项目/从0到1 | 创始人 + 秘密 + 幂律分布 |
| **彼得·林奇** | 成长股/中市值/行业轮动 | 十倍股 + PEG + 行业特工 |
| **达利欧** | 宏观策略/全天候/风险平价 | 周期 + 风险平价 + 去杠杆 |
| **卡尔·伊坎** | Activist投资/并购催化 | 催化因子 + 股东激活 + 估值差 |
| **德鲁肯米勒** | 全球宏观/趋势交易/外汇大宗 | 宏观趋势 + 仓位管理 + 止损纪律 |

---

## 🎯 支持的投资阶段

| 阶段 | 特征 | 推荐风格 |
|------|------|---------|
| **Pre-Seed / 天使** | 想法阶段，0收入 | 蒂尔框架 |
| **种子轮** | MVP验证，早期用户 | 蒂尔 + 林奇 |
| **A轮** | 单位经济验证 | 蒂尔 + 巴菲特（财务验证） |
| **B轮** | 规模化增长 | 林奇 + 巴菲特 |
| **成长期** | 快速扩张 | 林奇为主 |
| **成熟期** | 稳定盈利 | 巴菲特为主 |
| **二级市场** | 公开交易 | 巴菲特 + 达利欧 |
| **Late-Stage/Pre-IPO** | 上市前最后融资 | 巴菲特 + 伊坎 |
| **困境/重组** | 经营困难/破产边缘 | 特殊情境框架 |

## 🌐 支持的赛道

| 赛道 | 特点 | 特有分析维度 |
|------|------|-------------|

| **工业/制造** | 资本密集 | 产能利用率、资本回报 |
| **金融** | 监管驱动 | 杠杆、流动性风险 |

## 📋 分析流程（自动路由）

```
Step 1: 识别输入
├── 公司/项目名称
├── 股票代码（如有）
├── 投资风格偏好（如有）
├── 投资阶段
└── 赛道

Step 2: 路由匹配
├── 二级市场 + 价值偏好 → 巴菲特框架
├── 早期VC + 创始人评估 → 蒂尔框架
├── 成长股 + 行业轮动 → 林奇框架
├── 宏观策略 + 风险平价 → 达利欧框架
└── Activist + 估值差 → 伊坎框架

Step 3: 数据采集
├── 最新财报/融资材料
├── 行业数据
├── 竞争格局
└── 宏观环境

Step 4: 分析输出
└── 路由至对应框架的完整分析
```
