# 彼得·蒂尔投资框架（Peter Thiel VC Methodology）

### 早期风险投资 · "从0到1" · 幂律分布

---

## 核心哲学

> *"从0到1"不是从1到n的竞争，而是创造全新的垄断。*

蒂尔的核心信仰：
1. **幂律分布** —— VC回报不是正态分布，而是极少数项目贡献几乎所有回报
2. **秘密** —— 好公司建立在少数人知道而大多数不相信的秘密之上
3. **垄断** —— 目标是10年后占据整个市场，不是分一杯羹
4. **时机** —— 不是每个人都能在2009年买比特币，时机决定一切

---

## 蒂尔分析框架（7大模块）

### 模块1：创始人深度评估（最核心）

| 维度 | 问题 | 信号 |
|------|------|------|
| **认知能力** | 能否清晰表达"秘密"？ | 能用简洁语言解释复杂机会 |
| **战略深度** | 是长期博弈还是短期优化？ | 谈论10年而非下个季度 |
| **创始人组合** | 创始人之间关系动态？ | 能力互补+信任深厚 |
| **Founder Vesting** | 股权结构是否健康？ | 1年悬崖+4年全vest |
| **多次创业** | 之前是否创过业？ | 成功/失败经验都有价值 |
| **信息密度** | 对行业的理解深度？ | 能回答任何刁钻问题 |
| **逆向思维** | 是否在不该自信的地方过度自信？ | 健康的不自信+执行力自信 |

**评分**：创始人评估占投资决策的**70%+**，早于市场/产品/财务

### 模块2："从0到1"评估

```
核心测试：
□ 这个公司是否在创造全新的东西？
□ vs "1→n"——这个公司是复制粘贴别人，还是做别人做不到的事？
□ 这个机会是否在10年后消失？（如果是，说明不够"从0到1"）
□ 监管/社会是否准备好接受这个范式转变？
□ 这个公司是否有可能在细分市场建立垄断？
```

**垄断评分**：
- 完全竞争市场 → 不投
- 即将被大公司复制的 → 不投
- 能够建立护城河的细分垄断 → 核心标的

### 模块3：市场时机分析

```
蒂尔检查（Thiel Check）：
□ 为什么是现在？而不是5年前或10年后？
□ 监管是否到达临界点？
□ 技术的S曲线是否进入快速增长阶段？
□ 社会/文化是否接受这个范式转变？
□ 这个时机是否可以重复？
```

**时机信号**：
- 比特币/区块链：2010年前后是正确时机，现在已是"1→n"
- AI：2017-2020年是正确时机，现在竞争激烈
- CRISPR基因编辑：现在正是时机

### 模块4：幂律分布下的投资策略

```
组合逻辑：
□ 单个项目期望值 = Σ(p_i × return_i)
□ 在幂律分布下，"选错"比"买贵"更致命
□ 分散10-15个项目即可（vs VC行业惯例20+）
□ 红杉/Benchmark真实回报：Top 1-2项目贡献80%+

项目筛选漏斗：
1. 海选：100个项目 → 30个深入了解
2. 尽调：30个 → 10个进入portfolio
3. 加码：每半年复盘，Top performer加倍
```

**回报预期**：
- 天使轮/种子：目标 100x
- A轮：目标 30-50x
- B轮：目标 10-20x
- 失败率容忍：70%项目归零

### 模块5：竞争分析与护城河

```
蒂尔竞争观：
"竞争是Loser的游戏。真正胜利的企业不是因为竞争而胜利。"

护城河来源（按重要性排序）：
1. 网络效应（最强，但需要从小开始）
2. 规模经济（需要快速规模化）
3. 品牌（最难建立）
4. 专利（技术护城河）
5. 转换成本（存量客户的锁定）

警告信号：
⚠️ 创始团队说"我们没有竞争对手"
⚠️ 需要持续烧钱维持竞争地位
⚠️ 市场规模太小，无法支撑垄断
```

### 模块6：Token/Crypto项目特有评估

```
Token经济模型：
□ Token的核心效用是什么？（证券型 vs 应用型）
□ Token持有者权利：分红？治理？费用折扣？
□ Token分配：团队/投资者/社区/生态 各占多少？
□ 解锁时间表：是否存在大量抛压？
□ 激励机制：持有vs抛售哪个更有利？

协议评估：
□ TVL（总锁仓量）vs 真实用户数
□ 协议收入 vs Token价值捕获
□ 安全性：审计历史 + 代码质量
□ 去中心化程度：真去中心化 vs 名义去中心
```

### 模块7：退出路径评估

```
蒂尔对退出的看法：
"IPO是成功的标志，不是目标。目标是建立持久的垄断企业。"

退出路径评估：
□ IPO：市场空间是否足够大到支撑上市公司？
□ 并购：谁会买？为什么？价格区间？
□ secondary：早期投资者是否可以流动性退出？
□ 持有：是否值得持有10年？

退出时机判断：
□ 市场需求是否真正验证？
□ 竞争格局是否明朗？
□ 创始人是否准备好IPO？
□ 市场窗口是否合适？
```

## VC特定估值方法

### 阶段对应估值逻辑

| 阶段 | 估值逻辑 | 核心指标 |
|------|---------|---------|
| **Pre-Seed** | "这个创始人值多少钱？" | 创始人背景 |
| **Seed** | 用户/产品MVP验证 | 早期用户增长、留存 |
| **A轮** | 单位经济模型 | LTV/CAC、NPS、留存曲线 |
| **B轮** | 收入规模化 | 收入增长率、毛利率、NPS |
| **C轮+** | 市场领导力 | 市场份额、竞争壁垒 |
| **Late-stage** | 标准二级估值 | EV/Revenue、EV/EBITDA |

### 天使轮估值锚点

```
天使轮估值区间（2024年参考）：
- 美国：$5-15M pre-money
- 中国：$8-20M pre-money
- 欧洲：$3-10M pre-money
- Crypto：$5-30M pre-money（看团队背景）

估值公式：
post-money = pre-money + 融资额
ownership = 融资额 / post-money

经验法则：
- 天使轮出让15-25%
- A轮出让20-30%
- B轮出让15-25%
```

### Token估值方法

```
Token估值指标：
□ FDV（Fully Diluted Valuation）= 当前价格 × 总供应量
□ MC（Market Cap）= 当前价格 × 流通量
□ FDV/Revenue：类比传统EV/Revenue
□ TVL/FDV：DeFi协议特有指标

Token估值判断：
□ FDV是否与项目阶段匹配？
□ 团队/投资者token何时解锁？
□ 是否有持续买盘支撑？
□ Token持有者结构是否健康？
```

## 尽职调查清单（蒂尔版）

### 创始人DD

```
□ 创始人简历：教育+工作+创业经历
□ 社交媒体足迹：Twitter/LinkedIn/Medium
□ 过往融资历史：估值变化？投资人是谁？
□ 创始人之间的关系：如何认识？合作多久？
□ 法律尽职调查：是否有未披露的诉讼？
□ 创始人信用调查：是否有欺诈历史？
□ 心理健康：创始人是否有稳定的心理状态？
□ 家庭情况：家庭是否支持创业？
□ 退出预期：创始人是否有一致的目标？
```

### 产品DD

```
□ 产品原型：Demo vs 正式产品
□ 用户数据：真实用户 vs 虚假账号
□ 产品路线图：是否清晰可执行？
□ 技术架构：代码质量、扩展性、安全性
□ IP：是否申请专利？是否有侵权风险？
□ 监管风险：产品是否合规？
```

### 市场DD

```
□ TAM（Total Addressable Market）
□ SAM（Serviceable Addressable Market）
□ SOM（Serviceable Obtainable Market）
□ 市场份额：现在是多少？10年后能到多少？
□ 市场增长率：是增量市场还是存量争夺？
□ 政策风险：监管是否会影响市场？
```

## 投资决策模板

### 蒂尔风格评分卡

| 标准 | 0分 | 1分 | 2分 | 3分 |
|------|-----|-----|-----|-----|
| **创始人质量** | 一般 | 良好 | 优秀 | 顶尖 |
| **从0到1程度** | 完全竞争 | 差异化 | 细分垄断潜力 | 全新品类 |
| **时机** | 太早/太晚 | 略早/略晚 | 正确时机 | 完美时机 |
| **市场规模** | <$1B | $1-10B | $10-100B | >$100B |
| **护城河** | 无 | 弱 | 中等 | 强 |
| **退出路径** | 不清晰 | 可能 | 清晰 | 极佳 |

**决策阈值**：
- ≥15分：强烈买入
- 12-14分：买入
- 8-11分：观望
- <8分：不投

## 与巴菲特框架的核心区别

| 维度 | 巴菲特 | 彼得·蒂尔 |
|------|--------|----------|
| **阶段** | 成熟企业 | 早期VC |
| **回报分布** | 正态分布 | 幂律分布 |
| **时间视野** | 永远 | 10-15年 |
| **失败观** | 规避错误 | 拥抱失败 |
| **分析核心** | 护城河+低估 | 创始人+秘密 |
| **尽调重点** | 财务指标 | 创始人深度 |
| **估值** | DCF | 一切基于人 |
| **竞争观** | 护城河抵御 | 垄断消灭竞争 |
| **目标回报** | 合理收益 | 十倍/百倍 |
