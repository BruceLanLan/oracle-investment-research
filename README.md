## 这是什么？

**投资神谕研报系统**是一个AI驱动的投资研究框架，能够根据不同的投资风格、赛道和阶段，自动匹配最适合的分析方法。

它的核心是一个**智能路由器**——输入一个公司/项目，它会自动识别：
- 是VC早期还是二级市场？
- 是价值投资还是成长股？
- 属于什么赛道？

然后路由到最合适的分析框架，输出专业的投资研究报告。

## 📊 支持的投资风格

| 风格 | 人物 | 核心特点 | 最适场景 |
|------|------|---------|---------|
| **巴菲特** ⭐ | 沃伦·巴菲特 | 护城河 + 所有者盈余 + 安全边际 | 成熟企业/二级市场/长期价值 |
| **彼得·蒂尔** ⭐ | Peter Thiel | 从0到1 + 创始人评估 + 幂律分布 | 早期VC/创业项目/Crypto |
| **彼得·林奇** | Peter Lynch | 十倍股 + PEG + 行业特工 | 成长股/中市值/行业轮动 |
| **达利欧** | Ray Dalio | 周期 + 风险平价 + 去杠杆 | 宏观策略/全天候/全球配置 |
| **卡尔·伊坎** | Carl Icahn | 催化因子 + 股东激活 + 估值差 | Activist投资/并购催化 |
| **德鲁肯米勒** | Stanley Druckenmiller | 宏观趋势 + 仓位管理 + 止损纪律 | 全球宏观/外汇大宗/加密趋势 |

## 🎯 支持的投资阶段

```
Pre-Seed      → 蒂尔框架（创始人+秘密）
    ↓
Seed          → 蒂尔 + 林奇（用户验证）
    ↓
A轮           → 蒂尔 + 巴菲特（单位经济）
    ↓
B轮           → 林奇 + 巴菲特（规模化）
    ↓
成长期        → 林奇为主（高增长）
    ↓
成熟期        → 巴菲特为主（护城河+分红）
    ↓
二级市场      → 巴菲特 + 达利欧
    ↓
Pre-IPO       → 巴菲特 + 伊坎
    ↓
困境/重组     → 特殊情境框架
```

## 🌐 支持的赛道

| **金融** | 杠杆率、流动性风险、监管合规 |
